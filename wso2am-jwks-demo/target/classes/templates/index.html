Sign In to the Application

This is a SpringBoot MVC Application

  1. This uses KeyCloak for Single-Sign-On
    • with OpenID-Connect
    • then receives an ID Token (JWT)
  2. An application exists on WSO2 API Manager
    • with client-key
    • and client-secret
  3. This application invokes /token API endpoint on WSO2 API Manager
    • to obtain an OAuth/Opaque Token
    • using Service Provider's credentials
    • and JWT Grant-type
    • with the ID Token returned by KeyCloak
  4. Then it can invoke APIs deployed on WSO2 API Manager
    • by submitting OAuth/Opaque Token
    • in the Authorization Header
